# 💈 Cutly

Cutly is a modern, responsive barber shop landing page built with **React** and **Tailwind CSS**. It follows a **mobile-first approach**, ensuring a smooth experience across all devices, from mobile to desktop.

The design focuses on simplicity, clarity, and a premium visual style suitable for a real barber shop brand.

---

## ✨ Features

- 📱 Mobile-first responsive design  
- ⚛️ Built with React  
- 🎨 Styled with Tailwind CSS  
- 🧭 Smooth navigation with section-based layout  
- 💇 Hero section with strong visual impact  
- ✂️ About section with brand storytelling  
- 💲 Services & pricing showcase (in progress)  
- 📞 Call-to-action for bookings  

---

## 📐 Design Approach

Cutly is being developed with a **mobile-first mindset**, meaning:

- Layout is optimized for small screens first  
- Scales naturally to tablet and desktop  
- Focus on readability and spacing  
- Minimal but premium UI style
